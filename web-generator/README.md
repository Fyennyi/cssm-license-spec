# CSSM License Web Generator

This is a simple web application to generate ready-to-use CSSM Unlimited License files.

## Usage

- Run locally:
  ```
  npm install
  npm run dev
  ```
- Open in browser as prompted.

## Output

Download a filled LICENSE-2.0.txt or LICENSE-1.0.txt with your data.

## Build as static site

To generate a static site for GitHub Pages:
```bash
npm run build
```
Take the `dist/` folder and deploy it to Pages.
