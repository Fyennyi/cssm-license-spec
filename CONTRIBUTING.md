# Contributing to CSSM Unlimited License

We welcome improvements, clarifications, and suggestions for the license texts and documentation.

## How to Contribute

- Propose changes via GitHub Issues or Pull Requests.
- Use clear English.
- Keep modifications compatible with the spirit of CSSM Unlimited License.

## Review Process

- All proposals are discussed publicly via the Fyennyi group.
- Major changes require consensus.
